Getting the model from the vpk

Use TitanfallVPKTool to open "Titanfall2\vpk\englishclient_mp_common.bsp.pak000_dir.vpk" and look for the model that you want to replace.

importing into Blender

In Blender add the blender source tools addon to your enabled addons.
When you have gotten the smd from the vpk open Blender and import it.
Also import the model that you want to replace it with.

Now that you have imported the models make sure that the new model is the same size as the one that you want to replace and if has an armature make it so that they have the same pose.

Now apply the armature pose to the model and remove the armature.
After this you parent it to the skeleton of the model you want to replace.
I recommend that you keep the model you want to replace a bit to the side instead of deleting it, so that you can see how you will need to weight paint the new model for it to work properly.
