Startpage Installation Tools Modding

Tools you will need for modding models into Titanfall 2

TitanfallVPKTool to get the models from Titanfall.

Crowbar to decompile and recompile the models.

Legion or Legion+ for getting the textures and texture paths.

3D modelling program(Used Blender, can also download from steam) for editing the model you want to get into Titanfall.
- For Blender you will also need to install blender source tools addon to import the models.
- And Blender SG_Shader to make viewing the materials easier.

Image editing program that supports dds images(Used GIMP) for converting textures.

Repak to pack the textures to work with Titanfall.

mdlshit to compile the model into a mod.
